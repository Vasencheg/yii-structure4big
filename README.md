# yii-structure4big
Основанная на yii-boilerplate стуктура приложения для больших и средних проектов.

### Отличия и особенности
* Установленные и сконфигурированные модули yii-user и rights
* Набор часто-используемых расширений:
 * withRelatedBehavior
 * YiiSeoBehavior
 * translitUrlBehavior
 * Image Component
 * ElFinder Widget
 * Imperavi-Redactor
 * Uploadify
 * File-upload Action
